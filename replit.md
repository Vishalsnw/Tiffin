# TiffinFlow Pro

A modern, subscription-based Tiffin Management SaaS for small home kitchen businesses.

## Features
- Smart Dashboard with daily meal analytics.
- Customer Management with subscription tracking.
- Automated expiry and renewal reminders.
- Responsive, mobile-first UI with Dark/Light mode support.

## Project Structure
- `src/`: React source code.
- `src/App.jsx`: Main application logic and UI.
- `vite.config.js`: Development server and build configuration.

## Development
```bash
npm install
npm run dev
```

## Production
The project is configured for Replit Autoscale deployment.
```bash
npm run build
npm run preview
```
